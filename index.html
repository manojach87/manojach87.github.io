About

I was first introduced to programming when I was around 14. Ever since writing my first program in BASIC, I have been obsessed with the idea of using software to solve real life problems. I believe that as programmers, we can change the lives of people around us for good using the knowledge we have.

As my first professional experience, I built a website using Java Server Pages, Java Servlets, HTML, Java Script and My SQL database for a small firm. Although my UI/UX programming days are far behind me, the job helped me understand the challenges in software development and the ways to overcome them.

I now wear multiple hats at work, moving between the roles of a Data Scientist, a Data Quality Specialist and a Data Engineer/ETL Developer at Charter Communications, which, by the way, is also called Charter Spectrum. Most recently, I have been working on building better data quality solutions using datascience to be able to find issues before they occur. I enjoy taking part in kaggle competitions because they challenge my thinking and my understanding of data.

I play soccer, love playing the guitar and sometimes draw. Most people agree that Data Science is more of an art than just programming. Art, music, sports, and programming challenge various aspects of my mind : creativity, organization, multi tasking, problem solving, team work and prioritization. More often than once, this challenge keeps me up at night with a never ending thirst to create simple, and beautiful yet powerful things to share with the world.
